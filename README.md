## Getting Started


1. **Clone the repository:**
   ```bash
   git clone https://github.com/shrvnkmr/shrvnkmr-portfolio.git
   ```


2. **Install dependencies:**
   ```bash
   cd shrvnkmr-portfolio
   npm install
   ```


3. **Run the development server:**
   ```bash
   npm run dev
   ```
